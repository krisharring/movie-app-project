HTTP Method Request URL Request body data format Response body data format
GET Get all movie lists '/movies' None Get data array of all movie lists in JSON format
GET Get data about a single movie title '/movies/:title' None Get an array of a single movie data in JSON format, based on title entered in the URL parameter. Example: { title: 'Lupin', director: ['Louis Leterrier', 'Marcela Said'], genre: ['Crime', 'Thriller', 'Mystery', 'Drama'], releaseYear: 2021 }
GET Get movie by genre '/movies/genre/:title' None Get an array of movie by genre in JSON format
GET Get movie director name '/movies/director/:name' None Get an array of a movie director data in JSON format, based on the director entered in the URL parameter
POST Allow clients to create account '/users' Clients' user information added to the server in a JSON format style A JSON format of the client's added information, included with a unique id and an extra successproof message
PUT Allow clients to update their client info '/users/:username' None A JSON format data with the updated result
POST Allow clients to add their favourite movies to their library '/users/:username/movies/:movieId' A unique id of the favorite movie will be added to the users list. Example:"/users/'Tony'/movies/:619aab110bb457bf0b0afda4" An array of the users data in JSON format, and an added unique movieid. Example: "username": "Tony", "email": "tony@gmail.com", "favoriteMovie": "619aab110bb457bf0b0afda4"
